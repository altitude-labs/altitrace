# SDK Documentation

TypeScript SDK for HyperEVM transaction simulation and tracing.

## Installation

```bash
bun add @altitrace/sdk viem
```

## Quick Start

```typescript
import { createClient } from '@altitrace/sdk'

const client = createClient.local() // or .production()

// Simulate transaction
const result = await client.simulate({
  from: '0x742d35Cc6634C0532925a3b8D86C4F5e573F7d5B',
  to: '0xA0b86a33E6441a8F9d6f4c13f8a39c3A7a4e8c',
  data: '0xa9059cbb...'
})

// Trace transaction
const trace = await client.trace({
  from: '0x742d35Cc6634C0532925a3b8D86C4F5e573F7d5B',
  to: '0xA0b86a33E6441a8F9d6f4c13f8a39c3A7a4e8c',
  data: '0xa9059cbb...'
})
```

## Core Functions

- **simulate()** - Simulate transactions with gas estimates
- **trace()** - Get execution traces with multiple tracers
- **accessList()** - Generate access lists for gas optimization
- **batch()** - Process multiple transactions

## Access List Optimization

```typescript
// Generate access list for gas optimization
const accessList = await client.accessList({
  from: '0x742d35Cc6634C0532925a3b8D86C4F5e573F7d5B',
  to: '0xA0b86a33E6441a8F9d6f4c13f8a39c3A7a4e8c',
  data: '0xa9059cbb...'
})

console.log('Access list:', accessList.accessList)
console.log('Gas estimate:', accessList.gasUsed)
```

## Builder API Methods

### Simulation Builder
- `call(tx)` - Add transaction
- `forAccount(address)` - Track asset changes
- `withAssetChanges(enabled)` - Enable change tracking
- `withStateOverrides(overrides)` - Override account states
- `withBlockOverrides(overrides)` - Override block parameters
- `atBlock(tag)` - Set execution context
- `execute()` - Run simulation

### Trace Builder
- `call(tx)` / `transaction(hash)` - Set trace target
- `withCallTracer(config)` - Call hierarchy tracing
- `withPrestateTracer(config)` - State change tracing
- `withStructLogger(config)` - Opcode-level tracing
- `with4ByteTracer()` - Function signature tracing
- `atBlock(tag)` - Set block context
- `execute()` - Run trace
