# Installation

Install the Altitrace SDK in your project.

## Package Manager

:::code-group

```bash [bun]
bun add @altitrace/sdk
```

```bash [npm]
npm install @altitrace/sdk
```

```bash [yarn]
yarn add @altitrace/sdk
```

```bash [pnpm]
pnpm add @altitrace/sdk
```

:::

## Requirements

- Node.js 18+ or Bun
- TypeScript 5.0+ (recommended)

## Verify Installation

```typescript
import { AltitraceClient } from '@altitrace/sdk'

const client = new AltitraceClient()
const health = await client.healthCheck()

console.log('Status:', health.status)
```

## TypeScript Configuration

The SDK includes TypeScript definitions. For optimal type safety:

```json [tsconfig.json]
{
  "compilerOptions": {
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "moduleResolution": "bundler"
  }
}
```
